
# Aetherix

**Tavily AI-Powered Web Search Android App**

A clean, modern Android application that brings the power of [Tavily](https://tavily.com) — an advanced AI search engine — directly to your mobile device.

---

## ✨ Features

- **Real-time AI Search**: Powered by Tavily's intelligent search API
- **AI Answer Generation**: Get concise, summarized answers with sources
- **Customizable Search**: Adjust depth (Basic/Advanced/Fast/Ultra-Fast), result count, and more
- **In-App Settings**: Full control over API key and search parameters without recompiling
- **Persistent Configuration**: Settings saved securely on device
- **Responsive UI**: Supports light/dark mode and orientation changes
- **Clean Results Display**: Formatted output with scores, sources, and metadata

---

## 🛠️ Tech Stack

- **Language**: Java
- **UI Framework**: Android XML layouts + AppCompat
- **Networking**: OkHttp + JSON parsing
- **Storage**: SharedPreferences (for settings and API key)
- **Search Engine**: [Tavily API](https://tavily.com)

---

## 🚀 Getting Started

### Prerequisites
- Android Studio (latest stable)
- Android device or emulator (API 21+)
- A free/paid [Tavily API Key](https://app.tavily.com)

### Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/DroidSpectre/aetherix.git
   ```

2. Open the project in Android Studio.

3. Open the app → Tap **⚙ Settings**.

4. Enter your Tavily API key and configure search preferences.

5. Start searching!

## 🔧 Configuration

All settings are managed in-app:
- **API Key** (required)
- Search Depth (Basic / Advanced / Fast / Ultra Fast)
- Max Results (5 / 10 / 15 / 20)
- Include AI Answer, Images, Raw Content, Favicons, etc.
- Default Topic, Time Range, Country

No `.env` file or hardcoded credentials are used.
